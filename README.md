# Portfolio X

A premium one-page portfolio website for **Akash Karthik P**, designed to present enterprise data engineering credibility with the polish of a modern SaaS landing page.

## Project Purpose

Portfolio X exists to clearly position Akash as a:

- Enterprise Data Engineer
- Azure Data Engineer
- ETL Modernization Specialist
- Cloud Migration Engineer
- Future Cloud Data Architect

The website is intentionally focused on trust, clarity, performance, and recruiter readability.

It is not intended to be flashy. Its quality should come from typography, spacing, semantic structure, accessibility, and execution discipline.

---

## Product Goals

- Communicate technical capability within 20 seconds
- Highlight enterprise-scale data engineering experience
- Emphasize modernization, migration, and reliability outcomes
- Provide a polished recruiter-ready digital presence
- Maintain strong performance, accessibility, and SEO

---

## Design Direction

Portfolio X takes inspiration from the restraint and polish of:

- Apple
- Stripe
- Linear
- Microsoft Fluent
- Databricks

### Visual Characteristics

- Premium dark theme
- Large typography
- Generous whitespace
- Minimal motion
- Enterprise SaaS tone
- Calm, modern presentation

### Explicitly Avoided

- Cyberpunk aesthetics
- Neon effects
- Gaming UI patterns
- Terminal-heavy design
- Over-animated interactions

---

## Technology Constraints

This project must be built with:

- HTML5
- CSS3
- Vanilla JavaScript

This project must not use:

- React
- Vue
- Angular
- Bootstrap
- Tailwind CSS
- Heavy front-end frameworks

---

## Quality Requirements

The final implementation should be:

- Semantic
- Accessible
- Responsive

- Production-quality

Aim for scores above 95 in:

- Performance
